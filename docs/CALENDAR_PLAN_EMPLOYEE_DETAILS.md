# Детальная информация по сотрудникам в календарном плане

## Описание изменений

Добавлена возможность просмотра полной информации по каждому сотруднику в календарном плане, включая график осмотров и требуемые анализы.

## Функциональность

### Для клиники (`/dashboard/clinic/calendar-plan`)

При клике на количество сотрудников в календарном плане открывается модальное окно со списком всех сотрудников, включенных в план.

### Для работодателя (`/dashboard/employer/calendar-plan`)

Аналогичная функциональность доступна для работодателя при просмотре календарных планов.

## Отображаемая информация о сотруднике

### Основная информация
- ФИО
- Должность
- Участок/объект
- ИИН
- Телефон (если указан)

### Стаж работы
- Общий стаж
- Стаж на текущей должности

### Вредные факторы
- Список всех вредных факторов, воздействующих на сотрудника
- Отображается в виде цветных меток

### График осмотров
- Дата последнего осмотра
- Дата следующего планового осмотра
- Форматирование дат в удобном для чтения виде

### Требуемые анализы
Стандартный набор:
- Общий анализ крови
- Общий анализ мочи
- Флюорография
- ЭКГ

Дополнительно:
- Специальные анализы в зависимости от вредных факторов

### Примечания
- Дополнительная информация о сотруднике (если указана)

## Интерфейс

### Кликабельные элементы
1. **Основной счетчик сотрудников** - отображается под информацией о плане
   - При наведении меняет цвет и показывает стрелку
   - При клике открывает модальное окно

2. **Счетчики в карточках участков** - для планов с несколькими участками
   - Отображаются с подчеркиванием
   - При клике открывают модальное окно со всеми сотрудниками плана

### Модальное окно
- Полноэкранное с прокруткой
- Адаптивный дизайн (responsive)
- Анимация появления карточек сотрудников
- Кнопка закрытия в заголовке и футере
- Счетчик общего количества сотрудников

## Стилизация

### Цветовая схема
- **Вредные факторы**: оранжевый (`orange-50`, `orange-600`)
- **График осмотров**: синий (`blue-50`, `blue-600`)
- **Требуемые анализы**: зеленый (`green-50`, `green-600`)
- **Примечания**: серый (`gray-50`)

### Адаптивность
- Мобильные устройства: одна колонка
- Планшеты и десктопы: две/три колонки
- Модальное окно: максимум 90% высоты экрана с прокруткой

## Использование

1. Откройте страницу календарного плана
2. Найдите нужный план в списке
3. Кликните на количество сотрудников (например, "6 сотрудников")
4. Просмотрите детальную информацию в открывшемся окне
5. Закройте окно кнопкой "Закрыть" или кликом на крестик

## Преимущества

- Быстрый доступ к информации о сотрудниках
- Наглядное отображение графика осмотров
- Видимость вредных факторов и требуемых анализов
- Удобная навигация без перехода на другие страницы
- Адаптивный дизайн для всех устройств
